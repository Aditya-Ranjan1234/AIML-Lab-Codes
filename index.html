How to Run the Streamlit App

To run the Streamlit app, follow these steps:

  1. Make sure you have Python installed (Python 3.7 or higher recommended)
  2. Open a terminal or command prompt
  3. Navigate to the streamlit_app directory
  4. Run the following commands:
    5. pip install -r requirements.txt
    streamlit run app.py
  5. Alternatively, you can run the provided scripts:
    • On Windows: Double-click run_app.bat
    • On macOS/Linux: Run ./run_app.sh in terminal
